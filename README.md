# Shahzeb Rizvi - Data Engineering Consultant CV

A responsive HTML-based resume website showcasing my professional experience, skills, and projects as a Data Engineering Consultant.

## Files Structure

- `index.html` - Main resume page with professional summary, skills, work experience, education, certifications, and projects
- `contact-me.html` - Contact page with address and contact form
- `hobbies.html` - Personal interests and hobbies
- `css/styles.css` - Styling for the website
- `images/` - Profile picture and other assets
- `resume_latex_code.txt.txt` - Original LaTeX resume source code

## Usage

Open `index.html` in any modern web browser to view the resume. The site is designed to be hosted on any web server or viewed locally.

## Privacy Note

For privacy and security, sensitive contact information like phone numbers and email addresses have been removed from public display. Contact can still be made through the provided form or professional networking platforms.
